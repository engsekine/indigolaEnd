環境設定

0.gitを追加する場合

git init
git config --global user.name "engsekine"
git config --global user.email "engsekine@gmail.com"
touch .gitignore

1.yarnダウンロード　
yarn init -y

2.devDependenciesダウンロード
yarn add webpack webpack-cli webpack-dev-server babel-loader @babel/core @babel/preset-env html-loader html-webpack-plugin style-loader sass sass-loader css-loader autoprefixer postcss postcss-loader --dev

3.dependenciesダウンロード

まとめたやつ
git init && git config --global user.name "engsekine" && git config --global user.email "engsekine@gmail.com" && yarn add webpack webpack-cli webpack-dev-server babel-loader @babel/core @babel/preset-env html-loader html-webpack-plugin style-loader sass sass-loader css-loader autoprefixer postcss postcss-loader --dev
